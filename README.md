# Entrenament dels herois

App web per generar sessions d'entrenament integral (mental + físic + espiritual) per a nens, amb pocs clics. Les dades es guarden a Firebase (Firestore) i es comparteixen entre tots els dispositius de la família amb un compte compartit.

**➡️ App en línia: https://jordicortescom.github.io/entrenament-dels-herois/**

- **Disseny**: [generador-sessions-brief.md](generador-sessions-brief.md)
- **Pla de treball**: [ROADMAP.md](ROADMAP.md)
- **Tests d'invariants**: obre l'app amb la ruta [`#/tests`](https://jordicortescom.github.io/entrenament-dels-herois/#/tests)

## Ús

Obre l'[app](https://jordicortescom.github.io/entrenament-dels-herois/), entra amb el compte familiar, configura les dates de naixement dels fills a Paràmetres, tria durada, lloc i intensitat, i prem «Genera sessió». Al mòbil, pots afegir-la a la pantalla d'inici per usar-la com una app.
