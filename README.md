# Entrenament dels herois

App web per generar sessions d'entrenament integral (mental + físic + espiritual) per a nens, amb pocs clics. Un sol fitxer HTML, sense dependències ni servidor: les dades viuen al `localStorage` del navegador.

- **Disseny**: [generador-sessions-brief.md](generador-sessions-brief.md)
- **Pla de treball**: [ROADMAP.md](ROADMAP.md)
- **Tests d'invariants**: obre l'app amb la ruta `#/tests`

## Ús

Obre `index.html` en un navegador (o la versió publicada a GitHub Pages), configura les dates de naixement dels fills a Paràmetres, tria durada, lloc i intensitat, i prem «Genera sessió».
